## Installation
Clone the project
```
git clone https://github.com/onderakbulut/cine-app.git
```
Go to the project directory
```
cd cine-app
```
Install dependencies
```
npm install
```
Start the server
```
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
